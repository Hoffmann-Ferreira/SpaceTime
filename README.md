### ðŸ”§ Installation

## Clone repository

```bash
# command
$ git clone

# enter
$  git clone https://github.com/Hoffmann-Ferreira/SpaceTime.git
```

## Installation

```bash
Installation must be done folder by folder.
npm install
```

## Running the app

```bash
# Server
$ npm run dev

# Web
$ npm run dev

# Mobile
$ npm run start
```
